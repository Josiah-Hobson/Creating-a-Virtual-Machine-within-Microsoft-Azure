# Description
I'm going to show the steps (after you create a Azure Account) to make a Virtual Machine within Microsoft Azure. Creating a VM in Azure is a very important resource to know for businesses as well as people wanting to experiment with certain aspects of software installation without compromising an actual system.  
# Languages and Utilities Used
Microsoft Azure 
# Environments Used
Windows 10 (22H2)
